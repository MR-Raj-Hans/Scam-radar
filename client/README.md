# 💻 ScamRadar - Frontend

This directory contains the frontend source code for the ScamRadar platform. It is built using **React** and **Vite**, with **Tailwind CSS** for styling.

## 🚀 Development

1. Make sure the backend server is running (check the root directory for instructions).
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## 🛠️ Key Technologies
- **Vite:** Next-generation frontend tooling.
- **Tailwind CSS:** Utility-first CSS framework.
- **Framer Motion:** For smooth UI transitions and animations.
- **Lucide React:** A clean and consistent icon set.

For full project documentation and setup instructions, please refer to the [main README](../README.md).
